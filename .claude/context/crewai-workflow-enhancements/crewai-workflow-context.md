# Контекст чата: Улучшения workflow CrewAI

## Обсуждённые изменения:
1. **Удаление merger сервиса**: Удалён из кода Boss, так как не используется. Теперь Boss сам объединяет результаты Managers.
2. **Переход на GitHub репозитории**: Вместо zip — клонировать/пушить изменения в GH-репы для тестирования и хранения.
3. **Гибкие роли воркеров**: Воркеры общаются, пишут код, тестируют в зависимости от ролей/задач.
4. **Восстановление проекта**: Boss восстанавливает проект из Redis JSON (5-7 мин) перед присвоением Managers/Workers.
5. **Программное создание файлов**: Добавить поле "commands" в JSON воркеров для выполнения bash-команд в проекте (создание/редактирование файлов).
6. **Контекст в проекте**: Хранить в .crewai/context.json вместо БД для полноценной работы.
7. **Решение конфликтов**: Git merge вместо Redis locks — воркеры в ветках, merge с разрешением.

## Реализация:
- Обновить код Boss, Manager, Worker.
- Интегрировать Git для merge.
- Пересобрать Docker, исправить ошибки.
